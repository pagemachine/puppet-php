Function Details

(rvalue) to_hash_settings

This function converts a {key => value} hash into a nested hash and adds an id to the outer key.

Examples:

to_hash_settings({'a' => 1, 'b' => 2})

Would return:

{
  'a' => {'value' => 1},
  'b' => {'value' => 2}
}
